---------------------------------------------------------
[Usuario] "Necesito crear un sistema web monolítico en ASP.NET WebForms con C#. Tiene que tener Capa de Datos, Capa de Negocio y Presentación. Usa SQL Server para la base de datos."

[Usuario] "Crea las tablas principales: tbl_usuario, tbl_producto, tbl_categoria y tbl_proveedor. Usa LINQ to SQL (DBML) para la conexión de base de datos."

FASE 2: Seguridad Avanzada (Login, Roles y 2FA)
---------------------------------------------------------
[Usuario] "Haz una pantalla de Login y otra de Registro de usuarios. Asegúrate de encriptar todas las contraseñas en la base de datos usando AES."

[Usuario] "Agrégale validación obligatoria de cédula ecuatoriana al registrarse."

[Usuario] "Añade un sistema de roles. El ID '1' será Administrador y el '2' será Usuario normal. Dependiendo del rol, oculta las opciones del menú lateral en el Site.Master."

[Usuario] "Necesitamos elevar la seguridad: añádele autenticación de dos pasos (2FA). Al poner su clave, que le mande un código numérico (OTP) al correo para poder entrar al sistema."

[Usuario] "Añade un módulo para recuperar la contraseña olvidada. Que genere un token temporal y se lo mande por correo con un link."

FASE 3: Módulos de Mantenimiento (CRUD)
---------------------------------------------------------
[Usuario] "Crea el CRUD completo (Listar, Crear, Editar, Desactivar) para las Categorías y para los Proveedores. Todo usando procedimientos almacenados o LINQ."

[Usuario] "Ahora haz el CRUD para los Productos. Este tiene que ser más complejo: permite subir múltiples fotografías por producto. Guarda el archivo en la carpeta 'Uploads' y registra la ruta en otra tabla de base de datos (tbl_imagen_producto)."

[Usuario] "En las tablas donde listo la información (GridViews), agrégales una barra de búsqueda inteligente y un paginador numérico para que no se sature la pantalla."

FASE 4: Extras (Juegos e Importación Excel)
---------------------------------------------------------
[Usuario] "Haz una pantalla para que el administrador pueda subir muchos productos de golpe importando un archivo Excel (.xlsx o .csv). Haz que primero muestre una tabla de previsualización antes de guardar."

[Usuario] "Para el lado de los usuarios normales, agrégales algo entretenido: crea un par de juegos web interactivos como un Juego de Memoria y El Ahorcado."
